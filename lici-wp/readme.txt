LiveInternet crossposting plugin

== Description ==

LiveInternet crossposter (LIci WP) is a WordPress plugin, which automatically copies new posts to one or more LiveInternet blog accounts. 

== Installation ==

1. Upload `lici_wp/lici_wp.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Customize your options on the Settings->LIci WP page.
